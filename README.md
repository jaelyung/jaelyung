## 👨‍💻 About Me

My journey in software started with backend development using Java and Spring from 2020 to 2022 while working on government-funded projects. That experience is why many of the repositories here are written in Java.

In 2023, I joined Apple Developer Academy @ POSTECH and began exploring iOS development more deeply.

I enjoy building mobile apps that can become part of people's everyday lives. iOS development feels particularly appealing because it allows developers to implement Apple's design philosophy directly into real products.

These days, I'm building apps and experimenting with ideas to make language learning simpler and more accessible.
